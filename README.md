## Extension Settings

- `hatenablogger.hatenaId`: your Hatena ID
- `hatenablogger.blogId`: [your Blog ID](http://blog.hatena.ne.jp/my/config)
- `hatenablogger.apiKey`: [your API Key](http://blog.hatena.ne.jp/my/config/detail)
